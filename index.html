SAP S/4HANA 구축 모의프로젝트

SAP Cloud On Development Excellence #1 · 2026.03 ~ 진행 중

프로젝트 개요

SAP 코리아에서 실제 컨설팅을 진행하였던 고객사의 사례를 기반으로 진행한 모의 프로젝트입니다. 여러가지 주제 중 가구 판매 및 렌탈 서비스가 사업 모델인 기업을 골랐고, 주요 상품을 침대로 설정하였습니다. 저는 프로젝트 리더(PL)와 MM 모듈을 담당하여 구매 프로세스 전반을 ABAP과 Fiori로 구현하였습니다.

담당 역할

  • 프로젝트 리더(PL)로서 일정 관리 및 기술적 의사결정 주도
  • MM(자재관리) 모듈의 구매 프로세스 설계 및 개발 전담
  • 프로젝트 통일성을 위한 공통 컴포넌트 설계

주요 개발 내용

  • 원자재 이동평균가 및 리드타임 실시간 갱신 Module Pool 프로그램 개발
  • 구매 요청부터 송장 검증까지 MM 모듈 End-to-End 프로세스 구현
  • 생산 오더와 연계한 자동/수동 구매요청 생성 및 구매오더 전환 Report 프로그램 개발
  • 플랜트 간 Two-Step으로 이루어지는 완제품/반제품 재고 이전 Report 프로그램 개발
  • SCM 모듈에서 공통적으로 사용 가능한 자재 문서 생성 Function Module 및 자재 문서 관리 및 역분개 Report 프로그램 개발
  • Top-Of-Page를 대체하는 HTML Viewer 기반 프로그램 헤더 공통 컴포넌트

사용 기술

  • ABAP · CDS View · RAP · OData 기반 백엔드 개발
  • SAP Fiori Freestyle를 활용한 프론트엔드 UI 구현
